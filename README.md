# 3D 贪吃蛇网页游戏

## 快速开始

1. 在项目根目录启动本地静态服务器（任选其一）：
   - 使用 Python：
     ```bash
     python3 -m http.server 8000
     ```
   - 或使用 Node.js（如果已安装）：
     ```bash
     npx serve .
     ```
2. 打开浏览器访问 `http://localhost:8000`（或所选服务器输出的端口）。
3. 页面加载后，即可看到 3D 贪吃蛇游戏界面。使用键盘方向键控制蛇体移动，按下空格键可暂停或恢复游戏。

> 直接双击 `index.html` 可能会因为浏览器的跨域限制导致资源无法正常加载，因此推荐通过本地服务器启动。

## 项目结构

- `index.html`：页面主体结构，包含画布和 HUD。
- `style.css`：页面样式与玻璃质感叠层效果。
- `script.js`：Three.js 游戏逻辑，包括场景、控制、碰撞检测等。

## 依赖

项目直接引用 CDN 版本的 Three.js，无需额外安装依赖。
